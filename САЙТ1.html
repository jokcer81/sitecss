Обучение CSS

CSS (Cascading Style Sheets) - это язык стилей, который используется для оформления веб-страниц. Он позволяет задавать цвет, шрифт, размер, отступы и другие свойства элементов HTML.

В этом учебнике вы узнаете, как использовать CSS для создания красивых и удобных в использовании веб-страниц.

Селекторы CSS

Селекторы CSS - это инструмент, с помощью которого можно выбрать элементы HTML, чтобы применить к ним стили.

Примеры селекторов:

p { color: red; }

- селектор элемента, выбирает все элементы <p> и задает им цвет текста красный.

#myId { background-color: blue; }

- селектор идентификатора, выбирает элемент с идентификатором "myId" и задает ему фон синий.

.myClass { font-size: 16px; }

- селектор класса, выбирает все элементы с классом "myClass" и задает им размер шрифта 16px.

a[href="https://www.example.com"] { color: green; }

- селектор атрибута, выбирает все ссылки с атрибутом "href" со значением "https://www.example.com" и задает им цвет текста зеленый.
